Detailed Description

Example use-case scenario for hosting MagmaCoreJenaDatabase on a Fuseki server.

The FusekiService class can be used to host in-memory or persistent Magma Core Jena Datasets over HTTP using a Fuseki server.

By default, the Fuseki server is configured to run on localhost:3330, however this can be change by setting the

port(int)

.

The Fuseki server can host either in-memory Datasets, or connected TDB stores. Datasets can be added to the server using the

add(name, dataset)

method. Datasets are hosted at

localhost:<port>/<name>

.
