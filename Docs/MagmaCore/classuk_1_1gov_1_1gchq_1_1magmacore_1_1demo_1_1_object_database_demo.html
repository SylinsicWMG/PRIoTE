Detailed Description

Example use-case scenario for MagmaCoreObjectDatabase.

This example demo creates an in-memory MagmaCoreObjectDatabase populated with the ExampleDataObjects as HQDM Java objects.

PersonB1_Bob

can be queried for using the MagmaCoreObjectDatabase#findByPredicateIriAndStringValue(uk.gov.gchq.hqdm.iri.IRI, String) method. The resulting object(s) of this query are output to the command-line as RDF triples.
