Detailed Description

Example use-case scenario for MagmaCoreJenaDatabase.

This example demo creates an in-memory MagmaCoreJenaDatabase populated with the ExampleDataObjects as RDF triples.

The Jena dataset is transactional, so MagmaCoreJenaDatabase#begin() must be called before any operations can be performed on the dataset, including queries. Once complete, MagmaCoreJenaDatabase#commit() or MagmaCoreJenaDatabase#abort() should be called to finish a transaction.

PersonB1_Bob

can be queried for using the MagmaCoreJenaDatabase#findByPredicateIriAndStringValue(uk.gov.gchq.hqdm.iri.IRI, String) method. The resulting object(s) of this query are output to the command-line as RDF triples.
